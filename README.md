# GHD Stores

A modern e-commerce storefront built with vanilla HTML, CSS, and JavaScript.

## Features

- Responsive design
- Product catalog
- Shopping cart with localStorage
- Product pages
- Category filtering
- Checkout page
- Contact page

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)

## Running

Simply open `index.html` in your browser.

Or enable GitHub Pages to host it online.
